# the_traitors_fun
A fun made web app for the game to improve experience and facilitate the game.

the App will start the game asking for the number of players 3 - 8.
then will ask number of turns that will be played.


the app consiste in buttons for the three different events, banishment, murderer and silent night with possibility to the user to choice between them.
Also when a murderer event happens all votes are based in the number of players with the option to choose a number of "F" for faithful. The system will count the votes and display only the first number voted by the murderer.
displaying a question for the number voted: has shield? if the answer is no, then the number is dead and the player is out of the game.
When a banishment event happens the system will count the number of players that still in the game and will give an opportunity to everyone to vote, the system will display the numbers still in the game and count the number with more votes and eliminate the player from the game.
when only two players is in the game system, the system will automatically finished the turn, displaying a message finished turn. players count the result and can start the next turn.
